# Інструкція з встановлення та запуску

## Крок 1: Встановлення залежностей

```bash
# Створіть віртуальне середовище
python -m venv venv

# Активуйте віртуальне середовище
# macOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Встановіть залежності
pip install -r requirements.txt
```

## Крок 2: Налаштування бази даних

```bash
# Застосуйте міграції
python manage.py migrate

# Створіть суперкористувача (адміністратора)
python manage.py createsuperuser
```

## Крок 3: Створення тестових даних (опціонально)

```bash
# Створить категорії, теги та тестові статті
python manage.py create_sample_data
```

## Крок 4: Запуск сервера

```bash
python manage.py runserver
```

Сайт буде доступний за адресою: http://127.0.0.1:8000/

Адмін-панель: http://127.0.0.1:8000/admin/

## Основні функції

✅ **Моделі даних:**
- Category (категорії контенту)
- Tag (теги)
- AuthorProfile (профілі авторів)
- Article (статті з повною метаінформацією)
- NewsletterSubscriber (підписники)

✅ **Сторінки:**
- Головна з блоками контенту
- Список всіх статей
- Детальна сторінка статті
- Сторінки категорій
- Сторінки тегів
- Сторінки авторів
- Пошук

✅ **Функціонал:**
- Темна тема
- Live search (пошук в реальному часі)
- Lazy loading зображень
- Адаптивний дизайн
- SEO оптимізація (meta tags, Open Graph, sitemap)
- Кешування
- Пагінація
- Соціальні кнопки поділіться

✅ **Безпека:**
- CSRF захист
- Валідація даних
- Обмеження розміру завантажених файлів

## Наступні кроки для розробки

1. **Додати WYSIWYG редактор** для контенту статей (вже включено django-ckeditor в requirements.txt)
2. **Налаштувати email розсилку** для NewsletterSubscriber
3. **Додати коментарі** до статей
4. **Реалізувати систему рейтингів** статей
5. **Додати RSS feed**
6. **Налаштувати CDN** для статичних файлів у продакшені
7. **Додати аналітику** (Google Analytics)

## Продакшен налаштування

Перед деплоєм обов'язково:

1. Змініть `SECRET_KEY` в `settings.py`
2. Встановіть `DEBUG = False`
3. Додайте `ALLOWED_HOSTS`
4. Налаштуйте базу даних (PostgreSQL рекомендовано)
5. Налаштуйте статичні файли (whitenoise або CDN)
6. Налаштуйте медіа файли (S3 або інший storage)
7. Налаштуйте email backend
8. Додайте SSL сертифікат
